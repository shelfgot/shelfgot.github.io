Shlomi Helfgot built the Torah Library add on as a Free add on. This SERVICE is provided by Shlomi Helfgot at no cost and is intended for use as is.

Information Collection and Use

I value your trust in allowing this Service access to the Google Document API and the Google External Request API. The add on will use the former to access the user's active document -- and that active document alone -- in order to insert the sources requested by the user. The requested material is sent as a query (which includes none of the user's personal information whatsoever) to a third party, Sefaria (Sefaria.org), through the External Request API, which then returns the desired source material for insertion into the given active document. No other information is acessed from the user's Google account, and absolutely no personal information is retained or stored in any format whatsoever. I (Shlomi Helfgot) nor any other third party are privy to any user's information in any format.

Thus, the Torah Library add on's use and transfer to any other app of information received from Google APIs will adhere to Google API Services User Data Policy, including the Limited Use requirements
